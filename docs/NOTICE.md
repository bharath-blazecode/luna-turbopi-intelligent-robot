# Notice

## Academic Context

LUNA was originally developed as a two-person university project (ITD102 Mini-Project) at QUT College in 2025, completed by Bharath "Barry" Sampath and Zhirui Lu.

## What This Repository Is

This repository is a cleaned and restructured portfolio version of that work. It is intended to demonstrate technical skills and document the engineering process — not to serve as a copy or resubmission of the original assessment.

The following are **not** included in this repository:

- The original university submission document
- Student ID numbers or tutor details
- Assessment cover sheets or marking criteria
- Unedited report text or raw submission material

## What Has Been Changed

- Source code has been sanitized: private IP addresses, hardcoded local paths, informal comments, and assessment-specific wording have been removed or replaced
- Documentation has been rewritten as technical case study content rather than copied from the original report
- File and folder structure has been reorganised to reflect standard project conventions
